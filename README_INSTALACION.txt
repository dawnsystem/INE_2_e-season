INSTALACION Y USO
-----------------

1. NO REQUIERE INSTALACION
   - Este programa funciona directamente sin instalar nada
   - No necesita Python ni otros programas adicionales
   - Compatible con Windows 10 y Windows 11

2. COMO USAR
   - Hacer doble clic en INE_4_e-season.exe
   - Si Windows muestra un aviso de seguridad, hacer clic en "Mas informacion" 
     y luego en "Ejecutar de todos modos"
   - La aplicacion se abrira directamente

3. PRIMERA VEZ
   - Al iniciar por primera vez, aparecera un aviso legal
   - Leer y aceptar para continuar
   - Se creara automaticamente un archivo de historial

4. ARCHIVOS QUE SE CREAN
   - historial_exportaciones.json: Guarda el registro de todas las exportaciones
   - Archivos Excel generados: Se guardaran donde usted elija

5. FUNCIONES PRINCIPALES
   - Procesar archivos Excel del INE (semanales y mensuales)
   - Exportar datos procesados en formato INE
   - Mantener historial de todas las exportaciones
   - Recuperar datos de exportaciones anteriores

6. COMPATIBILIDAD
   - Diseñado para trabajar con archivos exportados desde E-Season
   - Procesa los formatos del INE para campings en Cataluña
   - Genera archivos listos para enviar al INE

7. COMPARTIR CON OTROS CAMPINGS
   - Puede enviar el archivo INE_4_e-season.exe a otros campings Capfun
   - Cada camping tendra su propio historial independiente
   - El archivo pesa aproximadamente 50-70 MB

8. SOPORTE
   - Para dudas sobre el uso, consultar el MANUAL_USUARIO.pdf
   - Para problemas tecnicos, contactar con el camping que distribuyo el programa

================================================================================

NOTAS IMPORTANTES:
- Guardar siempre una copia de seguridad de los archivos Excel originales
- Verificar los datos antes de enviarlos al INE
- El historial se actualiza automaticamente con cada exportacion
